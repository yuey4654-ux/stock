# 每日交易工作台

更新日期：2026-06-21

这个文件夹以后就是每天的唯一入口。日常只看这里，不需要再去翻 `reports`、`prediction_tracking`、`watchlists`。

## 每天打开顺序

1. 先看 `01_最新交易策略.md`
2. 盘前/盘中看 `02_明日预测票_2026-06-22.md`
3. 收盘后补 `03_每日预测台账.csv`
4. 复盘胜率看 `04_规则票复盘统计.md`
5. 如果要看全部预测，包括非规则票，看 `05_全量复盘统计.md`

## 文件说明

| 文件 | 用途 |
| --- | --- |
| `01_最新交易策略.md` | 当前执行规则，日常交易以这个为准 |
| `02_明日预测票_2026-06-22.md` | 最新一版预测票，旧预测不再作为明日统计口径 |
| `03_每日预测台账.csv` | 每只预测票的明细台账，收盘后补数据 |
| `04_规则票复盘统计.md` | 只统计按你的交易规则筛出来的票，用来计算主策略胜率 |
| `05_全量复盘统计.md` | 包含规则票和非规则热点票的总复盘 |
| `06_非规则票统计.md` | 不按交易规则推荐的热点/博弈票单独统计 |
| `07_当前股票池.md` | 当前观察池 |
| `08_最近持仓与预测复盘_2026-06-18.md` | 最近一次持仓和预测复盘 |
| `09_命中率判断规则.md` | 命中、部分命中、未命中的统计标准 |
| `历史明细/` | 完整旧版策略和后续需要留底的明细 |

## 当前统计口径

- 主策略胜率只看 `04_规则票复盘统计.md`。
- 非规则热点票只做参考，不直接修改你的主交易规则。
- 每天新增预测后，要写入 `03_每日预测台账.csv`。
- 每天收盘复盘后，要同步更新 `04_规则票复盘统计.md` 和 `05_全量复盘统计.md`。
