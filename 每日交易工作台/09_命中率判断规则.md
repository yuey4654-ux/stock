# 预测准确率判断规则

更新日期：2026-07-06
适用策略：稳健承接优先收紧策略 V2.1

## 2026-07-07 V2.1 分层命中规则

V2.1 的目标是提高正式规则票准确率，同时保留 4-5 只候选覆盖面。复盘时必须区分三层：

| 层级 | 是否计入正式规则票 | 复盘用途 |
| --- | --- | --- |
| 正式规则票 | 是 | 主策略胜率和策略调整依据 |
| 条件观察票 | 默认否；满足尾盘/收盘确认后可升级 | 保留机会覆盖，不直接拖累正式规则胜率 |
| 非规则观察票 | 否 | 记录题材热度和博弈质量 |

若 CSV 预测类型暂未新增“条件观察”字段，则在预测类型或复盘备注中明确写入“条件观察”。每日对比表仍按全量和规则票两套统计；策略判断优先看正式规则票。

### 条件观察票升级规则

条件观察票只有同时满足以下条件，才可在复盘中升级为正式规则票：

1. 所属板块尾盘仍有扩散，不能只有高标独涨。
2. 个股回踩承接区不破。
3. 14:30 后仍站回触发位或确认位。
4. 收盘不低于触发位或承接区上沿。
5. 未跌破降级线或深失效位。

盘中一度触发但收盘跌回，不得升级；最多记为“条件观察未兑现”。

### V2.1 结果标签补充

| 标签 | 用法 |
| --- | --- |
| 命中 | 正式规则票满足方向、买点、收盘确认，且未触发失效 |
| 部分命中 | 守住承接区但未收回触发位，或方向正确但执行点不完整 |
| 未命中 | 未触发、跌破承接区、收盘确认失败，或跌破失效 |
| 条件观察兑现 | 条件观察票满足升级条件，可在复盘备注记录，但不自动改写历史预测类型 |
| 条件观察未兑现 | 方向或盘中表现有热度，但未满足尾盘/收盘确认 |

## 2026-07-06信息前置复盘要求

从 2026-07-06 起，正式预测必须先有“信息前置快照”，再有预测票。复盘时除了判断个股是否命中，还要检查预测流程是否合规。

信息前置快照至少包含：

- 财联社重要快讯。
- 国内政策/监管/产业新闻。
- 海外隔夜新闻。
- 美股三大指数、纳指、费城半导体、大型科技股、AI芯片股表现。
- 美元、美债、油价、黄金、VIX 等风险偏好变量。
- A股板块涨幅、成交额、涨停扩散、龙头封单、冲高回落和个股承接质量。

若预测报告缺少完整信息快照，复盘备注必须标注“信息快照不完整”。这不直接改变个股命中/未命中结果，但会影响策略质量评价，并触发下一次预测降权。

新闻政策只能用于热点优先级，不能替代买点、承接区、失效位和收盘确认。若个股因新闻利好上涨但没有给出可执行买点，最多记部分命中；若跌破失效位，即使新闻逻辑仍在，也必须记未命中或失效。

## 2026-07-04策略回退说明

当前命中率判断口径退回到 2026-07-01 的“稳健观察优先版收紧策略”。2026-07-03 的“连续失准防守模式”不再作为默认复盘口径，只在极端风险环境下临时启用。

回退依据：2026年1-5月逐票回测中，旧策略合计严格命中率 31.7%、调整后命中率 47.0%；防守版最新策略合计严格命中率 23.4%、调整后命中率 41.6%。因此默认统计、排序和复盘继续以旧策略为主。

## 预测类型

| 类型 | 判断重点 | 命中条件 |
| --- | --- | --- |
| 核心承接 | 趋势票能否回踩不破、继续维持强势 | 当日不跌破失效位，且收盘涨幅为正或明显强于同类；收盘重新站回关键触发位 |
| 稳健观察 | 稳健票是否保持趋势 | 收盘守住关键均线/支撑，走势不弱于市场；若只守住但不转强，最多记部分命中 |
| 弹性进攻 | 高弹性票能否放量转强 | 收盘仍在触发区上方，且没有高开转弱、明显长上影或收盘跌回触发区 |
| 风险降级 | 被提醒风险的票是否继续偏弱 | 当日走弱、跌破关键位，或明显弱于同题材 |
| 轮动备选 | 非主线方向是否被市场轮到 | 题材整体走强且个股同步转强，否则不纳入主规则执行 |
| 非规则票 | 题材温度、博弈样本 | 写入每日预测台账，参与全量命中率；不纳入规则票策略调整依据 |

## 2026-07-01后新增策略判断口径

历史倒推显示，稳健观察的调整后命中率显著高于核心承接和弹性进攻。因此后续复盘除了统计全量命中率，还必须单独观察三类票质量：

- 稳健观察：作为主策略质量核心指标，调整后命中率目标不低于 65%。
- 核心承接：若连续 2 次出现盘中触发但收盘站不住，下一交易日自动降权。
- 弹性进攻：若连续 2 次冲高回落或长上影，下一交易日不得作为规则内正式票。
- 正式预测排序：不再默认把高弹性或强趋势票排在最前，优先按稳健承接质量排序。

## 结果标签

| 标签 | 含义 |
| --- | --- |
| 命中 | 走势与预测方向、触发条件基本一致，且没有触发失效 |
| 部分命中 | 方向大致对，但强度、收盘站位或触发点不够理想 |
| 未命中 | 与预测方向明显相反，或触发后快速失效 |
| 待复盘 | 尚未补收盘数据 |

## 收紧模式硬规则

以下情况不能记为命中：

1. 盘中突破，但收盘跌回触发区下沿。
2. 高开超过 3%，但收盘低于开盘价。
3. 盘中跌破失效位且放量，即使尾盘收回，也只能降级观察。
4. 弹性进攻票出现明显长上影，且收盘不在当日振幅上半区。
5. 核心承接票收盘为负，且没有明显强于同类板块。
6. 核心承接票近端过热，次日只靠继续加速触发，但没有回踩承接确认。
7. 弹性进攻票未获得板块扩散配合，只是个股单独冲高。

以下情况最多记为部分命中：

1. 守住关键支撑，但没有转强。
2. 方向正确，但未给出可执行触发点。
3. 高开超预期导致没有合理买点。
4. 只满足盘中触发，不满足收盘确认。
5. 稳健观察守住支撑但明显弱于同题材龙头。
6. 核心承接方向正确但高开超过 3%，且没有给出回踩买点。

## 统计口径

- 全量严格命中率 = 所有写入每日预测台账的票中，命中数量 / 已复盘数量。规则票和非规则票都参与。
- 全量调整后命中率 = (命中数量 + 0.5 * 部分命中数量) / 已复盘数量。规则票和非规则票都参与。
- 全量加权命中率：核心承接权重 1.5，稳健观察权重 1.0，弹性进攻权重 0.8，非规则票和其他类型默认权重 1.0。
- 规则票命中率另行统计，只用于判断主交易规则是否需要收紧或放宽。
- 非规则票可以单独统计博弈池质量，但不得从全量命中率中剔除。
- 策略有效性判断优先看稳健观察调整后命中率、规则票调整后命中率和核心承接失效率，而不是只看全量严格命中率。
- 若稳健观察调整后命中率高于 65%，但核心承接/弹性进攻拖累全量命中率，应优先调整票型占比，而不是否定全部策略。

## 策略调整信号

以下任一情况触发策略调整提醒：

1. 规则票严格命中率低于 60%。
2. 规则票调整后加权命中率低于 65%。
3. 规则票连续 2 次同类题材预测未命中。
4. 核心承接票连续 2 次失效。
5. 弹性票连续出现高开冲高回落。
6. 风险降级票反而连续走强。
7. 稳健观察调整后命中率低于 60%，说明主策略稳定性失效。
8. 核心承接占规则票比例超过 40%，且当日规则票命中率低于 60%。
9. 弹性进攻占规则票比例超过 20%，且当日出现冲高回落样本。

触发后，下一交易日默认进入收紧模式：减少新增票、降低弹性仓位、提高收盘确认要求。
