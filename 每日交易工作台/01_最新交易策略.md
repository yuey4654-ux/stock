# 最新交易策略

更新日期：2026-06-21  
适用市场：A股、港股  
当前打法：行业轮动 + 趋势波段 + 每日预测复盘统计

## 一句话原则

只做 `向上趋势 + 有成交 + 有催化 + 有触发条件` 的票；没有触发就不硬做，跌破失效位先处理。

## 仓位纪律

| 项目 | 当前规则 |
| --- | --- |
| 目标持仓数 | 4-6 只 |
| 默认结构 | 65% 稳健仓 + 35% 波段仓 |
| 单票上限 | 稳健仓不超过 20%，波段仓不超过 15% |
| 单一行业上限 | 不超过 30% |
| 单一题材上限 | 不超过 25% |
| 账户回撤 -5% | 总仓位下调 10%-20% |
| 账户回撤 -8% | 进入防守模式，停止新增进攻仓 |

## 买入前必须满足

1. 股票在当前主线里：光通信/算力液冷、AI硬件、半导体链、机器人/高端制造优先。
2. 有清楚的触发条件，例如站稳关键位、回踩不破、放量突破。
3. 有明确失效条件，例如跌破某个支撑且放量走弱。
4. 不能因为“感觉要涨”买，必须能写进预测台账。

## 站稳定义

`站稳` 至少满足下面 3 条中的 2 条，且第 1 条必选：

1. 收盘价高于目标位 1.5% 以上。
2. 连续 2 个交易日收盘价都在目标位之上。
3. 收盘价重新站上 MA5 和 MA10，且 MA5 >= MA10。

盘中冲上去但收盘没站上，不算站稳。

## 放量突破定义

1. 收盘价高于关键平台上沿 1.5% 以上。
2. 当日成交量 / 20日均量 >= 1.5。
3. 收盘在当日振幅上半区。
4. 当天涨幅 A股尽量不超过 9%，港股可放宽到 12%。

## 当前规则调整

来自最近复盘统计：

- 核心承接票不能只看“守住支撑”，还要看收盘是否翻红或明显强于同类。
- 弹性进攻票不能只看盘中冲高，必须看收盘站位。
- 前一日大涨后的票，第二天只做承接确认，不追高开。
- 非规则热点票单独统计，不纳入主交易规则胜率。
- 蓝思科技H 2026-06-18 数据已修正为命中，后续复盘必须核对收盘价口径。

## 每日流程

| 时间 | 动作 |
| --- | --- |
| 盘前 | 只看最新预测票和触发条件 |
| 盘中 | 未触发不买；高开不追；跌破失效位先处理 |
| 收盘后 | 更新预测台账，判定命中/部分命中/未命中 |
| 每周 | 看规则票命中率，如果连续走低再调整策略 |

## 策略调整触发

出现下面情况，需要提醒并调整交易模式：

1. 规则票连续两个交易日严格命中率低于 60%。
2. 规则票加权命中率低于 65%。
3. 同一类型票连续误判，例如高弹性票连续冲高回落。
4. 持仓弱票持续拖累组合，但仍在补仓摊平。
