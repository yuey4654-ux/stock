# 最新交易策略

更新日期：2026-07-04  
适用市场：A股、港股  
当前模式：稳健观察优先版收紧策略（以提高预测准确率为第一目标，非规则票纳入全量统计）

## 一句话原则

只做 `主线趋势 + 稳健承接 + 收盘确认 + 不追高开 + 失效先处理` 的票。预测准确率优先于弹性空间；稳健观察票优先进入正式预测，核心承接必须经过二次确认，弹性进攻默认降级为小仓/非规则观察。盘中冲高不等于触发，收盘站不住就不能升级；非规则博弈票不参与主策略胜率判断，但只要被列为预测票，就必须进入每日台账和全量命中率统计。

## 当前策略状态

2026-07-04 已从“连续失准防守模式”回退到本策略。回退原因：2026年1-5月新旧策略逐票回测显示，防守版最新策略虽然把出票数从 564 条压到 154 条，但严格命中率从 31.7% 降至 23.4%，调整后命中率从 47.0% 降至 41.6%。因此后续默认不再使用“连续失准防守模式”作为主策略，只把它作为极端风险环境下的临时降级开关。

当前执行基准：

- 默认使用“稳健观察优先版收紧策略”。
- 每日规则票以 3-5 只为宜，不机械补满。
- 稳健观察优先，核心承接限量，弹性进攻严格降权。
- 若市场主线退潮、外盘科技大跌、规则票连续失效，可临时降到 1-2 只，但不能长期替代主策略。

2026-06-22 复盘后进入 `收紧模式`，原因：

- 规则票严格命中率 57.1%，低于 60% 阈值。
- 规则票严格加权命中率 55.4%，调整后加权命中率 62.2%，低于 65% 阈值。
- 主要亏分来自高开冲高回落、核心承接盘中跌破失效位、港股弹性票收盘跌回触发区。

退出收紧模式的条件：

- 后续连续 2 个交易日规则票严格命中率重新站上 65%；且
- 调整后加权命中率重新站上 70%。

## 2026年历史倒推后的策略结论

新增日期：2026-07-01

用 2026-01-05 至 2026-06-12 的 623 条历史倒推票验证，整体严格命中率 31.5%，调整后命中率 46.6%。分类型结果显示：

| 类型 | 样本数 | 严格命中率 | 调整后命中率 | 策略含义 |
| --- | ---: | ---: | ---: | --- |
| 稳健观察 | 210 | 45.7% | 67.4% | 主仓和正式预测优先类型 |
| 核心承接 | 315 | 26.3% | 37.8% | 必须增加过热过滤和二次确认 |
| 弹性进攻 | 98 | 17.3% | 30.6% | 不再作为常规规则票主推 |

排名回测也显示，原本靠前的强趋势票准确率偏低，排名 4-5 的稳健样本调整后命中率分别为 65.7% 和 69.0%。因此从下一次正式预测开始，排序不再按弹性和涨幅优先，而按“稳健观察优先、确认质量优先、过热风险扣分”重新排序。

## 三层执行闸门

### 1. 规则闸门

只允许以下类型进入主策略执行池：

- 稳健观察
- 核心承接（二次确认后）
- 弹性进攻

正式预测票结构必须满足：

- 每日规则内正式预测以 3-5 只为宜。
- 稳健观察不少于规则票数量的 50%，优先排在前两档。
- 核心承接最多 1-2 只，且必须通过过热过滤。
- 弹性进攻最多 1 只；若市场环境不是强势主升，默认改为非规则观察。
- 如果没有足够稳健观察票，宁愿减少预测数量，不用核心承接或弹性票补满。

以下类型不直接作为主策略买入依据，但如果当天被明确推荐/预测，必须写入每日预测台账并参与全量命中率统计：

- 非规则票
- 非规则高风险观察
- 纯题材进攻
- 轮动备选中没有板块确认的票

### 2. 高开闸门

高开不是买点，必须先看承接。

| 情况 | 执行规则 |
| --- | --- |
| 高开小于 3% | 可按原触发条件观察，但仍需守住关键位 |
| 高开 3%-6% | 前 30 分钟不追；只有回踩不破触发区、重新转强，才允许半仓试错 |
| 高开超过 6% | 默认不追，只允许尾盘收盘确认后列入次日观察 |
| 高开后收盘低于开盘 | 判定为追涨失败样本，不升级 |
| 高开突破但收盘跌回触发区下沿 | 盘中触发作废，按未命中或失效处理 |

### 3. 失效闸门

失效位优先级高于题材、涨幅和主观判断。

- 盘中跌破失效位且放量：直接降级，不补仓。
- 收盘跌破失效位：判定失效，不再用“尾盘收回一点”做安慰。
- 核心承接票若收盘为负，除非明显强于同类且未破关键位，否则最多只算观察，不升级。
- 弹性进攻票若出现高开转弱、长上影、收盘不在日内振幅上半区，不追，次日重新确认。

## 买入前必须满足

1. 在当前主线内：光通信、算力液冷、AI硬件、半导体链、机器人/高端制造优先。
2. 有清楚触发条件：站稳关键位、回踩不破、放量突破、收盘确认。
3. 有明确失效条件：跌破某个支撑并放量走弱。
4. 不因“感觉要涨”买入，必须能写进预测台账。
5. 收紧模式下，不能因为非规则票走强而放宽规则票条件。
6. 正式预测优先选择稳健观察票；若候选票只能靠次日继续加速才命中，默认降级。
7. 若大盘或主线板块当日冲高回落，次日不新增弹性进攻票。

## 过热过滤规则

以下任一情况出现，不能直接作为核心承接正式主推，最多降级为观察：

- 近 10 个交易日涨幅超过 20%，但最近 2 日没有缩量横盘或回踩承接。
- 5 日成交额均值超过 20 日成交额均值 2.2 倍，且收盘未创有效新高。
- 收盘处于近 20 日区间 85% 以上，同时出现长上影或收盘低于开盘。
- 前一交易日涨幅超过 6%，次日只允许看回踩承接，不作为开盘追入票。
- 同题材已有 2 只以上核心承接票当日冲高回落，次日该题材只保留稳健观察。

通过过热过滤后的核心承接，仍必须满足“回踩不破 + 重新站回触发位 + 收盘确认”，否则不算正式执行信号。

## 分类型执行规则

### 核心承接

适用：已经在主线中走强，次日看承接质量的票。

历史倒推显示核心承接严格命中率只有 26.3%，因此核心承接不再默认排第一。只有同时满足主线强度、非过热、承接区清晰、失效位近且明确，才允许进入正式规则票。

必须满足：

- 不跌破失效位。
- 收盘涨幅为正，或明显强于同类板块。
- 收盘站回关键触发位，而不是只看盘中上冲。
- 前一日没有明显高位爆量长上影。
- 近 10 日涨幅过大时，必须先出现 1-2 日缩量承接或回踩不破。

降级条件：

- 收盘为负且未明显强于同类。
- 盘中跌破失效位并放量。
- 高开后回落，收盘没有重新站回触发区。

### 稳健观察

适用：趋势较稳、弹性较弱、用于观察板块强弱的票。

历史倒推显示稳健观察调整后命中率 67.4%，是当前主策略优先类型。正式预测时，稳健观察优先排在前列，承担提高准确率的任务。

必须满足：

- 守住关键均线或支撑。
- 走势不弱于市场或同类板块。
- 没有明显放量长阴或连续冲高回落。
- 最好处于趋势中段，而不是连续加速末端。
- 买点必须靠近支撑或回踩区，不能靠追高突破。

收紧模式下，稳健观察票不承担进攻任务；如果只守住但不转强，最多记部分命中。正式推荐时，稳健观察的买点优先级高于核心承接的追涨突破点。

### 弹性进攻

适用：短线弹性强、可能继续突破的票。

历史倒推显示弹性进攻调整后命中率只有 30.6%，不再作为常规主推类型。除非市场进入明显强势主升，且题材有板块涨停扩散，否则弹性进攻默认放入非规则观察或小仓博弈。

必须满足：

- 收盘仍在触发区上方。
- 收盘位于当日振幅上半区，或至少没有明显高开转弱。
- 高开超过 3% 时，不追开盘价，只等回踩确认或尾盘确认。

降级条件：

- 高开后收盘低于开盘。
- 长上影且收盘不在振幅上半区。
- 盘中突破但收盘跌回触发区下沿。

## 仓位纪律

| 项目 | 收紧模式规则 |
| --- | --- |
| 目标持仓数 | 3-5 只 |
| 默认结构 | 60%-70% 稳健观察 + 20%-30% 核心承接 + 0%-10% 弹性 |
| 单票上限 | 稳健不超过 18%，核心不超过 15%，弹性不超过 6%-8% |
| 单一行业上限 | 不超过 30% |
| 单一题材上限 | 不超过 25% |
| 新开仓节奏 | 同一天最多新增 1-2 只规则票 |
| 账户回撤 -5% | 总仓位下调 10%-20% |
| 账户回撤 -8% | 防守模式，停止新增进攻仓 |

## 每日流程

| 时间 | 动作 |
| --- | --- |
| 盘前 | 只看最新预测票、触发区、失效位；标记高开风险票 |
| 开盘后 30 分钟 | 高开票不追，先看是否守住触发区和开盘价 |
| 盘中 | 未触发不买；跌破失效位先处理；非规则预测票也要按触发/失效记录 |
| 尾盘 | 优先看收盘站位，确认是否能升级为执行票 |
| 收盘后 | 更新预测台账，复盘命中/部分命中/未命中 |

## 正式预测排序规则

从 2026-07-01 起，正式预测票按以下顺序排序：

1. 先选稳健观察：主线明确、回踩区清楚、不过热、失效位近。
2. 再选核心承接：必须通过过热过滤，并且有明确二次确认买点。
3. 最后选弹性进攻：只在强势题材扩散时给 0-1 只，否则列入非规则观察。
4. 如果排名靠前的票严格命中率持续低于排名 4-5，不再按涨幅或弹性排前，而按预测准确性排前。
5. 每只票必须写清“不买条件”，尤其是高开超过 3%、开盘后30分钟承接失败、收盘站不回触发位。

## 策略调整触发

出现下面情况，继续保持或加强收紧模式：

1. 规则票连续两个交易日严格命中率低于 60%。
2. 规则票加权命中率低于 65%。
3. 同一类型票连续误判，例如高弹性票连续冲高回落。
4. 核心承接票连续两次盘中跌破失效位。
5. 风险降级票反而连续走强，说明风险过滤过度，需要单独复核。

## 本次调整验证

用 2026-06-15 至 2026-06-22 的 33 条规则票回放：

- 原规则：严格命中率 57.6%，调整后命中率 68.2%。
- 新过滤后可执行样本：25 条，严格命中率 76.0%，调整后命中率 84.0%。
- 过滤掉 8 条高风险样本，其中规避 5 条未命中，未过滤掉已命中样本。

结论：新规则主要提高的是执行质量，不是提高预测数量。代价是部分“方向对但确认不足”的样本会被降级观察；这符合当前收紧模式。

## 时政热点前置过滤规则

新增日期：2026-06-25

从下一次生成预测票开始，所有正式预测票都必须先经过时政热点前置过滤。时政热点只决定“优先看哪些板块、哪些票进入候选池”，不能替代买点、承接和失效规则。

更新日期：2026-07-02

每日预测必须新增“财联社 + 国内政策 + 海外新闻 + 美股表现”四项前置核验。预测票不能只根据A股技术形态生成，必须先判断隔夜和当日新闻是否会改变风险偏好、热点优先级和买点要求。

### 每日必须扫描

| 类别 | 观察重点 | 用途 |
| --- | --- | --- |
| 国际新闻 | 美股科技、AI芯片、半导体、美元、美债、油价、地缘风险 | 判断外盘映射和风险偏好 |
| 国内政策 | 国务院、发改委、工信部、商务部、能源局、交易所政策 | 判断政策催化方向 |
| 产业事件 | 财报、订单、涨价、产能、技术突破、行业会议 | 判断题材持续性 |
| A股盘面 | 板块涨幅、成交额、涨停扩散、龙头封单、冲高回落 | 判断资金是否真正认可 |
| 财联社快讯 | 盘中和盘后重要快讯、政策解读、产业链异动、公司公告摘要 | 捕捉当日最直接的题材催化和风险信号 |
| 美股表现 | 道指、标普、纳指、费城半导体、英伟达、AMD、博通、台积电、苹果、微软、Meta、特斯拉等 | 判断AI、半导体、科技成长方向的外盘映射 |
| 风险偏好变量 | 美债收益率、美元指数、油价、黄金、VIX、地缘风险 | 判断A股成长题材能否承接风险偏好 |

### 热点评分

| 维度 | 分值 | 判断方式 |
| --- | ---: | --- |
| 政策强度 | 0-25 | 是否有部委/国家级政策，是否明确指向产业链 |
| 国际映射 | 0-20 | 海外同类资产是否走强或有财报催化 |
| 盘面验证 | 0-30 | 当日是否放量、是否有龙头、是否有板块扩散 |
| 个股承接 | 0-15 | 龙头是否回踩不破，是否收盘站稳 |
| 风险扣分 | -20-0 | 是否高开过热、长上影、监管风险、连续加速 |

执行标准：

- 80分以上：可作为明日主线候选。
- 65-79分：只选龙头或稳健样本。
- 50-64分：只观察，不进入正式预测票。
- 低于50分：不纳入明日预测。

### 硬约束

时政热点只能加分，不能豁免以下条件：

- 高开超过 3% 不追。
- 必须有明确回踩承接区。
- 必须有失效位。
- 核心承接必须收盘站稳触发位。
- 弹性进攻必须收在日内振幅上半区，或回踩后重新转强。

### 新选票顺序

1. 先做财联社快讯、国际新闻、美股表现、国内政策、产业事件和A股盘面扫描。
2. 给热点板块排序，选出 3-5 个可能主线。
3. 若美股科技/半导体明显走弱，或A股同题材出现批量跌停/破位，规则票自动降数量，不得强行补满。
4. 从高分板块中挑龙头、趋势承接票和稳健观察票；弹性票只在强势主升时保留。
5. 每只票必须写清热点来源、买点、失效位和不买条件。
6. 预测报告必须包含“新闻政策与美股影响”小节，说明利好/利空板块、A股映射方向、以及对规则票数量和买点的影响。
7. 收盘复盘时，额外记录热点判断是否兑现。
