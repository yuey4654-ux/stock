# 每日交易工作台：最新交易策略

当前版本：稳健承接与逆向纪律策略 V3
更新日期：2026-07-19

完整策略以 [`prediction_tracking/latest_trading_strategy.md`](../prediction_tracking/latest_trading_strategy.md) 为唯一基准。

## 每日核心顺序

1. 信息前置快照。
2. 市场环境与板块扩散。
3. 标的资格与三层出票。
4. 触发位、承接区和失效位。
5. 单票及组合仓位上限。
6. 基准价阶梯加减仓。
7. 尾盘确认与收盘复盘。

## V3 盘面纪律速查

- 早盘快跌看承接，早盘直线拉升不追。
- 午后暴涨不追，午后大跌不抢，等次日确认。
- 趋势混沌时空仓休息。
- 不追高、不盲目割肉；震荡区间中部不交易。
- 阴线靠近支撑择机买，阳线达到计划区择机卖。
- 逆情绪布局，不逆已确认的趋势破位。
- 高低位震荡耐心等信号。
- 高位横盘后再冲高，分批锁定利润并上移保护位。

## 阶梯仓位速查

统一以首次正式建仓成交均价为基准价，一轮交易不重置。

| 涨跌幅 | 动作 |
| --- | --- |
| -5% | 不机械操作，检查逻辑与失效位 |
| -15% | 条件全部通过时，增持计划最大仓位 10% |
| -25% | 仅限高质量中长期标的，复核后再增持 25% |
| +5% / +15% | 继续持有并上移保护位 |
| +25% | 累计卖出计划最大仓位 10% |
| +35% | 累计卖出 20% |
| +45% | 累计卖出 30% |
| +60% | 累计卖出 40% |
| +100% | 原则上清仓；长期核心资产需重新立项 |

基本面/监管风险、趋势失效位、账户回撤和仓位上限，均优先于阶梯规则。
