# 预测与复盘台账说明

目标：每天记录盘前/盘中给出的预测票，收盘后用统一口径复盘，后续统计预测准确率并调整交易模式。

## 文件

- `daily_predictions.csv`：主台账，每条记录是一只票的一次预测。
- `accuracy_rules.md`：命中率判断规则。

## 记录流程

1. 预测当天先写入：
   - 预测日期
   - 适用交易日
   - 标的、代码、市场
   - 预测层级
   - 预测逻辑
   - 触发条件
   - 失效条件
2. 收盘后补充：
   - 收盘价
   - 涨跌幅
   - 是否触发
   - 是否失效
   - 命中结果
   - 复盘备注
3. 每周统计：
   - 总命中率
   - 核心票命中率
   - 弹性票命中率
   - 风险提示命中率
   - 最容易误判的题材/形态
