# 全量自选池

更新日期：2026-06-30  
用途：保存所有曾经预测、观察、午后加入、用户新增和降级观察的标的。以后每日复盘时，除了复盘当天预测票，还要扫描本池并重新分层。

## 当前分层数量

| 层级 | 数量 |
| --- | ---: |
| 核心执行池 | 4 |
| 强势观察池 | 23 |
| 轮动观察池 | 41 |
| 低权重观察池 | 7 |
| 降级观察池 | 27 |

## 使用规则

1. 当天正式预测票继续写入 `03_每日预测台账.csv`，用于命中率统计。
2. 全量自选池每天复盘时全部扫一遍，用于发现之前观察票突然表现。
3. 触发提醒不等于买入，仍要看回踩承接、收盘确认和失效位。
4. 午盘推荐继续单独进 `12_午盘推荐台账.csv`，不和正式预测台账混用。

全量明细详见同目录：`10_全量自选池.csv`。
