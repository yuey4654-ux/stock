# 最新交易策略

更新日期：2026-06-22  
适用市场：A股、港股  
当前模式：收紧版趋势承接策略（规则票优先，非规则票只观察）

## 一句话原则

只做 `主线趋势 + 收盘确认 + 不追高开 + 失效先处理` 的票。盘中冲高不等于触发，收盘站不住就不能升级；非规则博弈票不参与主策略胜率判断。

## 当前策略状态

2026-06-22 复盘后进入 `收紧模式`，原因：

- 规则票严格命中率 57.1%，低于 60% 阈值。
- 规则票严格加权命中率 55.4%，调整后加权命中率 62.2%，低于 65% 阈值。
- 主要亏分来自高开冲高回落、核心承接盘中跌破失效位、港股弹性票收盘跌回触发区。

退出收紧模式的条件：

- 后续连续 2 个交易日规则票严格命中率重新站上 65%；且
- 调整后加权命中率重新站上 70%。

## 三层执行闸门

### 1. 规则闸门

只允许以下类型进入主策略执行池：

- 核心承接
- 稳健观察
- 弹性进攻

以下类型只做题材温度记录，不直接作为主策略买入依据：

- 非规则票
- 非规则高风险观察
- 纯题材进攻
- 轮动备选中没有板块确认的票

### 2. 高开闸门

高开不是买点，必须先看承接。

| 情况 | 执行规则 |
| --- | --- |
| 高开小于 3% | 可按原触发条件观察，但仍需守住关键位 |
| 高开 3%-6% | 前 30 分钟不追；只有回踩不破触发区、重新转强，才允许半仓试错 |
| 高开超过 6% | 默认不追，只允许尾盘收盘确认后列入次日观察 |
| 高开后收盘低于开盘 | 判定为追涨失败样本，不升级 |
| 高开突破但收盘跌回触发区下沿 | 盘中触发作废，按未命中或失效处理 |

### 3. 失效闸门

失效位优先级高于题材、涨幅和主观判断。

- 盘中跌破失效位且放量：直接降级，不补仓。
- 收盘跌破失效位：判定失效，不再用“尾盘收回一点”做安慰。
- 核心承接票若收盘为负，除非明显强于同类且未破关键位，否则最多只算观察，不升级。
- 弹性进攻票若出现高开转弱、长上影、收盘不在日内振幅上半区，不追，次日重新确认。

## 买入前必须满足

1. 在当前主线内：光通信、算力液冷、AI硬件、半导体链、机器人/高端制造优先。
2. 有清楚触发条件：站稳关键位、回踩不破、放量突破、收盘确认。
3. 有明确失效条件：跌破某个支撑并放量走弱。
4. 不因“感觉要涨”买入，必须能写进预测台账。
5. 收紧模式下，不能因为非规则票走强而放宽规则票条件。

## 分类型执行规则

### 核心承接

适用：已经在主线中走强，次日看承接质量的票。

必须满足：

- 不跌破失效位。
- 收盘涨幅为正，或明显强于同类板块。
- 收盘站回关键触发位，而不是只看盘中上冲。

降级条件：

- 收盘为负且未明显强于同类。
- 盘中跌破失效位并放量。
- 高开后回落，收盘没有重新站回触发区。

### 稳健观察

适用：趋势较稳、弹性较弱、用于观察板块强弱的票。

必须满足：

- 守住关键均线或支撑。
- 走势不弱于市场或同类板块。
- 没有明显放量长阴或连续冲高回落。

收紧模式下，稳健观察票不承担进攻任务；如果只守住但不转强，最多记部分命中。

### 弹性进攻

适用：短线弹性强、可能继续突破的票。

必须满足：

- 收盘仍在触发区上方。
- 收盘位于当日振幅上半区，或至少没有明显高开转弱。
- 高开超过 3% 时，不追开盘价，只等回踩确认或尾盘确认。

降级条件：

- 高开后收盘低于开盘。
- 长上影且收盘不在振幅上半区。
- 盘中突破但收盘跌回触发区下沿。

## 仓位纪律

| 项目 | 收紧模式规则 |
| --- | --- |
| 目标持仓数 | 3-5 只 |
| 默认结构 | 70% 稳健/核心承接 + 30% 弹性 |
| 单票上限 | 核心/稳健不超过 18%，弹性不超过 10% |
| 单一行业上限 | 不超过 30% |
| 单一题材上限 | 不超过 25% |
| 新开仓节奏 | 同一天最多新增 1-2 只规则票 |
| 账户回撤 -5% | 总仓位下调 10%-20% |
| 账户回撤 -8% | 防守模式，停止新增进攻仓 |

## 每日流程

| 时间 | 动作 |
| --- | --- |
| 盘前 | 只看最新预测票、触发区、失效位；标记高开风险票 |
| 开盘后 30 分钟 | 高开票不追，先看是否守住触发区和开盘价 |
| 盘中 | 未触发不买；跌破失效位先处理；非规则票只记录温度 |
| 尾盘 | 优先看收盘站位，确认是否能升级为执行票 |
| 收盘后 | 更新预测台账，复盘命中/部分命中/未命中 |

## 策略调整触发

出现下面情况，继续保持或加强收紧模式：

1. 规则票连续两个交易日严格命中率低于 60%。
2. 规则票加权命中率低于 65%。
3. 同一类型票连续误判，例如高弹性票连续冲高回落。
4. 核心承接票连续两次盘中跌破失效位。
5. 风险降级票反而连续走强，说明风险过滤过度，需要单独复核。

## 本次调整验证

用 2026-06-15 至 2026-06-22 的 33 条规则票回放：

- 原规则：严格命中率 57.6%，调整后命中率 68.2%。
- 新过滤后可执行样本：25 条，严格命中率 76.0%，调整后命中率 84.0%。
- 过滤掉 8 条高风险样本，其中规避 5 条未命中，未过滤掉已命中样本。

结论：新规则主要提高的是执行质量，不是提高预测数量。代价是部分“方向对但确认不足”的样本会被降级观察；这符合当前收紧模式。
