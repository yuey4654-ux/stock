# 港股预测准确率判断规则

更新日期：2026-07-08

## 统计范围

- 只统计 `prediction_tracking/hk_daily_predictions.csv`。
- `是否计入准确率=是` 的票全部进入港股全量准确率分母。
- 港股台账与 A 股 `daily_predictions.csv` 分开，不混算。
- 条件观察、非规则观察如果写入港股台账，也参与港股全量准确率。

## 命中标签

| 标签 | 口径 |
| --- | --- |
| 命中 | 满足触发/升级条件，收盘确认，且未触发失效条件 |
| 部分命中 | 方向正确或守住承接区，但未完成收盘触发确认 |
| 未命中 | 未触发、跌破承接区、收盘跌回触发位下方，或触发失效条件 |
| 待复盘 | 尚未补充目标日收盘数据 |

## 港股特别规则

- 港股高开超过 4% 默认不追，必须等30分钟承接。
- 盘中突破但收盘跌回触发位，不算命中。
- 弹性票冲高回落且收盘低于开盘，最多记部分命中，若跌破失效位则记未命中。
- 非规则观察票不作为主策略放宽依据，但不得从港股全量准确率中剔除。
