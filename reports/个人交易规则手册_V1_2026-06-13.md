# 个人交易规则手册 V1

更新日期：`2026-06-13`
适用市场：`A股`、`港股`
打法定位：`行业轮动 + 趋势波段`

---

## 1. 先定边界

这套规则不是为了抓最低点和最高点，而是为了做三件事：

1. 只做 `向上趋势 + 有成交 + 有催化` 的机会。
2. 把模糊判断改成 `可观察数据 + 固定阈值`。
3. 优先控制回撤，而不是追求每一笔都赚。

### 账户级硬规则

- 目标持仓数：`4-6 只`
- 默认结构：`65% 稳健仓 + 35% 波段仓`
- 账户回撤到阶段高点 `-5%`：总仓位下调 `10%-20%`
- 账户回撤到阶段高点 `-8%`：进入防守模式，停止新增进攻仓
- 单一行业总敞口上限：`30%`
- 单一题材总敞口上限：`25%`
- 单票仓位上限：
  - 稳健仓：`20%`
  - 波段仓：`15%`

## 2. 指标与定义

本手册后续所有规则，统一使用下面的指标定义。

### 2.1 均线

- `MA5`：5 日收盘均价
- `MA10`：10 日收盘均价
- `MA20`：20 日收盘均价
- `MA60`：60 日收盘均价

### 2.2 成交量

- `VOL`：当日成交量
- `VMA5`：5 日平均成交量
- `VMA20`：20 日平均成交量
- `量比`：盘中成交速度相对过去 5 日同时间段均值

### 2.3 波动与位置

- `20日区间高点`：过去 20 个交易日最高价
- `20日区间低点`：过去 20 个交易日最低价
- `20日区间位置` = `(收盘价 - 20日低点) / (20日高点 - 20日低点)`
- `偏离MA20` = `(收盘价 / MA20) - 1`
- `ATR14`：14 日真实波幅均值，用于判断是否“波动过大”

### 2.4 动能

- `RSI14`
- `MACD DIF`
- `MACD DEA`
- `MACD柱` = `DIF - DEA`

### 2.5 强弱

- `相对板块强度`：个股近 5 日涨跌幅 - 所属板块近 5 日涨跌幅
- `相对指数强度`：个股近 5 日涨跌幅 - 基准指数近 5 日涨跌幅

## 3. 模糊词量化定义

这部分是核心。以后你说“企稳”“放量”“突破”，都按这里执行。

### 3.1 什么叫“站稳”

`站稳` 必须同时满足下面 3 条中的至少 2 条，且第 1 条必选：

1. 收盘价高于目标位 `1.5%` 以上。
2. 连续 `2` 个交易日收盘价都在目标位之上。
3. 收盘价重新站上 `MA5` 和 `MA10`，且 `MA5 >= MA10`。

补充限制：

- 如果只是盘中冲上去，但收盘没有站上，不算站稳。
- 如果靠尾盘偷拉勉强站上，但成交量没有同步放大，不算高质量站稳。

### 3.2 什么叫“关键平台”

`关键平台` 默认只认 3 类：

1. 过去 `10-20` 个交易日内，至少 `3` 次回踩未破的横盘区间上沿。
2. 前一波放量突破后的回踩确认区。
3. `MA20`、`MA60` 与前期密集成交区重叠的位置。

如果一个位置只是“你感觉差不多”，但没有横盘、没有回踩、没有均线重叠，就不定义为关键平台。

### 3.3 什么叫“放量”

`放量` 默认分 3 档：

- 弱放量：`VOL / VMA20 >= 1.2`
- 标准放量：`VOL / VMA20 >= 1.5`
- 强放量：`VOL / VMA20 >= 2.0`

实际执行时：

- 突破日至少要达到 `标准放量`
- 回踩确认日可以接受 `缩量`，标准是 `VOL / VMA20 <= 0.9`

### 3.4 什么叫“放量突破”

`放量突破` 必须同时满足：

1. 收盘价高于关键平台上沿 `1.5%` 以上。
2. 当日 `VOL / VMA20 >= 1.5`。
3. 收盘价位于当日振幅上半区，最好接近当日最高价。
4. 当天涨幅不超过 `9%`。

第 4 条是为了避免把情绪加速板当成可复制买点。对港股可放宽到 `+12%`。

### 3.5 什么叫“回踩不破”

`回踩不破` 必须满足：

1. 回踩对象是 `MA10`、`MA20`、前高平台、突破位`其中之一`。
2. 盘中跌破后能收回，或全天最低价未有效跌穿支撑位 `1.5%`。
3. 回踩日成交量低于突破日，且 `VOL / VMA20 <= 0.9` 更佳。
4. 回踩后 `1-3` 个交易日内重新转强。

### 3.6 什么叫“止跌企稳”

`止跌企稳` 至少满足下面 4 条中的 3 条：

1. 连续下跌后出现缩量十字星、小阳线或下影线结构。
2. 收盘重新站上 `MA5` 或收复前一日高点。
3. `RSI14` 从 `35-45` 区间拐头向上。
4. 次日没有再创新低。

### 3.7 什么叫“趋势走坏”

满足下面任意 1 条，就视为趋势走坏：

1. 收盘跌破 `MA20` 且连续 `2` 日未收回。
2. 收盘跌破关键平台下沿 `1.5%` 以上。
3. 放量长阴，且 `VOL / VMA20 >= 1.5`，跌幅大于 `4%`。
4. `MA10` 向下跌破 `MA20`，同时股价处于 `MA20` 下方。

### 3.8 什么叫“量缩价稳”

同时满足：

1. `VOL / VMA20 <= 0.9`
2. 当日实体跌幅不超过 `2%`
3. 收盘仍在 `MA10` 或关键支撑位之上

### 3.9 什么叫“相对不弱”

满足下面任意 1 条：

1. 个股近 `5` 日涨幅高于所属板块 `3%` 以上。
2. 板块反弹时，个股涨幅位于板块核心跟踪池前 `30%`。
3. 指数下跌日，个股跌幅小于板块平均跌幅。

## 4. 交易模式拆分

你当前最适合的不是“什么都做”，而是只保留 2 个主模式。

### 模式 A：强势回踩再转强

这是你的主模式，优先级最高。

#### 前提条件

- 大盘没有系统性杀跌
- 所属板块位于当日或近 3 日强势方向
- 个股中期趋势未坏：`MA10 > MA20 > MA60`

#### 入场条件

1. 过去 `20` 日涨幅在 `8%-35%`
2. 回踩 `MA10 / MA20 / 突破平台` 之一
3. 回踩期间没有放量大阴线
4. 次日或 `3` 日内重新出现 `站稳 + 转强`
5. `RSI14` 位于 `45-70`
6. `DIF >= DEA`，或两者距离收敛即将金叉

#### 买入触发

- 收盘买入版：
  - 收盘重新站上 `MA5` 或突破回踩小平台
  - `VOL / VMA20 >= 1.2`
- 盘中买入版：
  - 上午不追
  - 下午确认不破回踩低点后再考虑

#### 风控

- 初始仓位：`5%-8%`
- 止损：
  - 跌破回踩低点 `1.5%-2%`
  - 或总亏损达到 `-5%到-7%`

### 模式 B：平台站稳后放量突破

这是第二模式，但优先级低于模式 A，因为更容易买在加速段。

#### 前提条件

- 平台整理时间不少于 `8` 个交易日
- 平台振幅最好小于 `15%`
- 突破前 3 日没有连续巨量长上影

#### 入场条件

1. 平台上沿明确
2. 收盘突破平台上沿 `1.5%` 以上
3. `VOL / VMA20 >= 1.5`
4. `RSI14 <= 75`
5. `偏离MA20 <= 12%`
6. 当天不是纯消息刺激后的首次高潮板

#### 买入触发

- 突破日尾盘买一部分：`3%-5%`
- 次日不破突破位并继续放量，再加：`3%-5%`

#### 风控

- 跌回平台内并连续 `2` 日收不回去，减仓或离场
- 假突破当日放量但收长上影，不开仓

## 5. 禁做清单

下面这些情况，默认不做，除非后面你专门开发第三套模式。

- 连续暴涨后第一次看到利好消息的票
- `RSI14 > 75` 且放巨量长上影
- 股价距离 `MA20` 超过 `15%`
- 放量下跌后的第一次弱反弹
- 日成交额太小，无法稳定进出
- 公告、财报、监管不确定性过大，但你又没有事件优势

### 流动性红线

- A股日成交额低于 `5亿`
- 港股日成交额低于 `1亿港币`

低于这个标准，不进入主观察池。

## 6. 仓位与加减仓规则

### 6.1 初始仓位

- 稳健仓：`10%-15%`
- 波段仓：`5%-8%`

### 6.2 加仓条件

只有满足下面任意 1 套，才允许加仓：

#### 加仓触发 A

- 首笔开仓后浮盈为正
- 标的再次出现 `站稳关键平台后放量突破`
- 板块同步走强

#### 加仓触发 B

- 首笔开仓后完成一次 `回踩不破`
- 回踩日量缩价稳
- 再次转强时 `VOL / VMA20 >= 1.2`

### 6.3 禁止加仓

- 首笔仓位还在浮亏
- 板块走弱
- 个股跌破失效位
- 只是因为“成本高了想摊低”

### 6.4 减仓规则

#### 保护利润减仓

满足任意 1 条，先减 `1/3`：

1. 到达前高或阶段压力位，但量能跟不上
2. `偏离MA20 >= 12%`
3. 近 `5` 日涨幅超过 `18%`
4. 单票仓位因上涨被动超上限

#### 确认离场

满足任意 1 条，剩余仓位退出：

1. 趋势走坏
2. 事件逻辑兑现后无新增催化
3. 板块退潮，个股相对强度连续 `3` 日转负

## 7. 每笔交易必须填写的字段

以后每一笔交易卡，都必须写清这 8 个字段：

1. 标的名称
2. 所属模式：`强势回踩 / 平台突破`
3. 触发条件
4. 买入价
5. 初始仓位
6. 加仓条件
7. 止损条件
8. 失效条件

如果这 8 个字段写不出来，就默认不下单。

## 8. 当前版本的交易模式声明

你当前的模式可以正式写成下面这句话：

`我主要做 A股和港股里有主线的强势板块，只做趋势向上、流动性足够、回踩企稳后再转强或平台整理后放量突破的标的；我不做左侧抄底，不做无量反弹，不做趋势走坏后的侥幸持有；我用固定仓位、固定止损和固定加减仓条件来控制账户回撤。`

## 9. 实施顺序

别一下子把所有规则都扩到 20 套，先按这个顺序来：

1. 先固定 `2 个主模式`
2. 再固定 `8-10 个关键指标定义`
3. 再把规则写进每日交易看板
4. 最后再做脚本自动判分
