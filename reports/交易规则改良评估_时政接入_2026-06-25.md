# 交易规则改良评估与时政热点接入建议_2026-06-25

## 一句话结论

改良后的交易规则方向是对的，但还没有到可以放宽的阶段。当前最有效的改良点是过滤高开追涨、收盘不确认和失效后硬扛；下一步可以接入时政热点，但只能作为选股前置过滤和权重评分，不能替代承接买点。

## 规则表现评估

统计口径：只看规则票，不把非规则热点票纳入主策略胜率。

## 是否符合预期

基本符合预期，但属于“执行质量改善”，不是“预测能力完全稳定”。

- 符合预期的部分：高开不追、收盘确认、失效先处理，确实减少了方向对但买点差的亏分样本。
- 仍不理想的部分：核心承接严格命中率偏低，说明强势票次日经常给的是冲高回落或弱承接，不适合简单按前一日强度排序。
- 最近三日调整后命中率仍只有 61.1%，说明规则还处在收紧验证期，不能因为某一天方向对就放宽。

## 主要问题

1. 核心承接票过于依赖前一日强度，缺少第二天板块继续发酵的外部催化确认。
2. 高开票虽然被过滤，但候选池仍容易集中在已经过热的票上。
3. 选股逻辑更多来自盘面强弱，对国际新闻、国内政策、产业催化的前置排序还不够系统。
4. 弹性进攻票命中率相对更高，但波动大，不能扩大仓位，只能作为小仓位进攻。

## 时政热点接入是否可行

可行，而且应该接入。

但接入位置必须清楚：时政热点用于决定“明天先看哪些板块”和“哪些票进入候选池”，不直接决定买入。最终买入仍然必须满足回踩承接、关键位站回、收盘确认和失效位不破。

## 建议新增规则

### 1. 时政热点前置过滤

每天生成预测票前，必须先完成四类信息扫描：

| 类别 | 观察重点 | 作用 |
| --- | --- | --- |
| 国际新闻 | 美股科技、AI芯片、半导体、美元、美债、油价、地缘风险 | 判断外盘映射和风险偏好 |
| 国内政策 | 国务院、发改委、工信部、商务部、能源局、交易所政策 | 判断政策催化方向 |
| 产业事件 | 财报、订单、涨价、产能、技术突破、行业会议 | 判断题材持续性 |
| A股盘面 | 板块涨幅、成交额、涨停扩散、龙头封单、冲高回落 | 判断资金是否真正认可 |

### 2. 热点评分

每个候选板块先打分，再决定是否从中选票。

| 维度 | 分值 | 判断方式 |
| --- | ---: | --- |
| 政策强度 | 0-25 | 是否有部委/国家级政策，是否明确指向产业链 |
| 国际映射 | 0-20 | 海外同类资产是否走强或有财报催化 |
| 盘面验证 | 0-30 | 当日是否放量、是否有龙头、是否有板块扩散 |
| 个股承接 | 0-15 | 龙头是否回踩不破，是否收盘站稳 |
| 风险扣分 | -20-0 | 是否高开过热、长上影、监管风险、连续加速 |

执行标准：

- 80分以上：可作为明日主线候选。
- 65-79分：只选龙头或稳健样本。
- 50-64分：只观察，不进入正式预测票。
- 低于50分：不纳入明日预测。

### 3. 个股仍需过原规则

时政热点只能加分，不能豁免以下条件：

- 高开超过3%不追。
- 必须有明确回踩承接区。
- 必须有失效位。
- 核心承接必须收盘站稳触发位。
- 弹性进攻必须收在日内振幅上半区或回踩后重新转强。

## 接入后的选票流程

1. 先做时政与国际新闻扫描。
2. 给热点板块排序，选出3-5个可能主线。
3. 用当天盘面验证热点是否有资金承认。
4. 从高分板块中挑龙头、趋势承接票、弹性票。
5. 每只票必须写清买点、失效位和不买条件。
6. 收盘复盘时，记录这只票来自哪个热点板块，并统计热点判断是否有效。

## 是否需要继续改进

需要，但不是继续加复杂买卖条件，而是补上“热点来源”和“板块评分”的记录。

下一步重点：

- 在每日预测票中新增“热点来源”和“板块评分”字段。
- 复盘时增加“热点判断是否兑现”。
- 连续跟踪至少5个交易日，再判断时政热点接入是否提高命中率。

## 当前执行建议

维持收紧模式，不放宽。

正式规则调整为：

主线趋势 + 时政热点前置过滤 + 盘面验证 + 回踩承接 + 收盘确认 + 失效先处理。

这套改法更适合你现在的交易目标：既跟上热点，又不被消息刺激带着追高。
