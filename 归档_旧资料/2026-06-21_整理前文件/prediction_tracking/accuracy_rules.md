# 预测准确率判断规则

## 预测类型

| 类型 | 判断重点 | 命中条件 |
| --- | --- | --- |
| 核心承接 | 趋势票能否回踩不破、继续维持强势 | 当日不跌破失效位，且收盘涨幅为正或明显强于同类 |
| 弹性进攻 | 高弹性票能否放量转强 | 当日涨幅明显跑赢指数，且没有冲高大幅回落 |
| 稳健观察 | 稳健票是否保持趋势 | 收盘守住关键均线/支撑，走势不弱于市场 |
| 风险降级 | 被提醒风险的票是否继续偏弱 | 当日走弱、跌破关键位，或明显弱于同题材 |
| 轮动备选 | 非主线方向是否被市场轮到 | 题材整体走强且个股同步转强 |

## 结果标签

| 标签 | 含义 |
| --- | --- |
| 命中 | 走势与预测方向、触发条件基本一致 |
| 部分命中 | 方向大致对，但强度或触发点不够理想 |
| 未命中 | 与预测方向明显相反，或触发后快速失效 |
| 待复盘 | 尚未补收盘数据 |

## 统计口径

- 总命中率 = 命中数量 / 已复盘数量
- 加权命中率：核心承接票权重 1.5，普通观察票权重 1.0，情绪票权重 0.8。
- 不把“没触发所以没买”算作错误；如果预测本身要求等待触发，收盘后按“是否给出有效触发机会”单独记录。
